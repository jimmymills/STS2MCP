## Common Misidentifications

| Often Confused | How to Tell Apart |
|----------------|-------------------|
| Meal Ticket vs Strawberry | Meal Ticket is a RED TICKET/VOUCHER, Strawberry is a FRUIT with seeds |
| War Paint vs Meal Ticket | War Paint has BRUSH STROKES/SPLATTER pattern |
| Strawberry vs Pear | Strawberry is RED, Pear is GREEN/YELLOW |
| Shuriken vs Old Coin | Shuriken has POINTED STAR shape, Coin is ROUND |
| Swift Potion vs Cunning Potion | Swift has FEATHER, Cunning has BLADE/SHIV |
| Strength Potion vs Fire Potion | Strength has FIST icon, Fire has FLAMES |
| Block Potion vs Dexterity Potion | Block has SHIELD, Dexterity has MOVEMENT lines |

## Shop Layout Reference

```
+------------------------------------------+
|  [Card] [Card] [Card] [Card] [Card]      |  <- 5 Cards for sale (top)
|  [Card] [Card]                           |  <- 2 Cards for sale (bottom)
|                                          |
|  [Relic1] [Relic2] [Relic3]             |  <- 3 Relics (RIGHT SIDE, TOP ROW)
|  [Potion1] [Potion2] [Potion3]          |  <- 3 Potions (RIGHT SIDE, BOTTOM ROW)
|                                          |
|  [Card Removal: Xg]                      |  <- Bottom option
+------------------------------------------+
```

Prices generally:
- Common potions: ~48-55g
- Uncommon potions: ~70-85g  
- Rare potions: ~90-120g
- Common relics: ~150-180g
- Uncommon relics: ~200-250g
- Rare/Shop relics: ~250-300g
