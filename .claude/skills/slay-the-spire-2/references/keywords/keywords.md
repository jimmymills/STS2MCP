# Slay the Spire 2 Keywords & Mechanics

## Buffs (Positive Effects)

| Keyword | Effect | Notes |
|---------|--------|-------|
| **Artifact** | Negates debuffs. Each stack negates one debuff application. | Strips before debuff applies. Very strong vs debuff-heavy enemies. |
| **Block** | Prevents damage until next turn. Removed at start of your turn. | Barricade makes it persist. |
| **Buffer** | Prevent the next time you would lose HP. | Blocks one instance of damage/HP loss completely. |
| **Dexterity** | Each point adds +1 Block to all block-gaining cards. | Permanent scaling. Very strong. |
| **Intangible** | Reduce ALL damage taken and HP loss to 1 this turn. | Incredibly powerful. Multi-hit attacks counter it. |
| **Plating** | Gain Block at end of turn. Reduced by 1 at start of turn. | Degrades over time. Worse than STS1 Metallicize. |
| **Regen** | Heal HP at end of your turn. Reduced by 1 each turn. | Stacking regen = big healing. |
| **Ritual** | Gain Strength at start of each turn. | Scaling strength over time. |
| **Strength** | Each point adds +1 damage to ALL attacks. | Core scaling stat. Multi-hit benefits most. |
| **Thorns** | When hit by an attack, deal damage back. | Passive damage. Great vs multi-hit enemies. |
| **Vigor** | Your next Attack deals additional damage. Then consumed. | One-time damage boost. |

## Debuffs (Negative Effects)

| Keyword | Effect | Notes |
|---------|--------|-------|
| **Confused** | Card costs randomized (0-3) when drawn. | From Snecko Eye. Can be good or bad. |
| **Frail** | Gain 25% less Block from cards. | Devastating for block-heavy turns. |
| **Poison** | Lose HP equal to stacks at start of turn. Reduced by 1 each turn. | Silent's main damage source. |
| **Vulnerable** | Take 50% more damage from Attacks. | Paper Phrog makes it 75%. Key debuff. |
| **Weak** | Deal 25% less damage with Attacks. | Reduces enemy threat significantly. |
| **Doom** | At end of enemy turn, if Doom ≥ HP, enemy dies. | Necrobinder mechanic. |

## Card Keywords

| Keyword | Effect | Notes |
|---------|--------|-------|
| **Ethereal** | If in hand at end of turn, Exhaust it. | Auto-exhausts if not played. |
| **Exhaust** | Remove from deck until end of combat. | Core Ironclad mechanic. Triggers Dark Embrace, Feel No Pain. |
| **Innate** | Start each combat with this card in hand. | Guarantees key cards turn 1. |
| **Retain** | Not discarded at end of turn. | Hold cards for perfect timing. |
| **Replay** | Play this card an additional time. | Doubling effect. Very powerful. |
| **Unplayable** | Cannot be played from hand. | Curses, status cards. Must exhaust or discard. |

Enchantments are **permanent** modifiers that attach to cards for the entire run. Found via events.

| Enchantment | Effect | Notes |
|-------------|--------|-------|
| **Adroit** | Gain additional Block when played. | +Block on defensive cards. |
| **Corrupted** | Deal 50% more damage, lose 2 HP when played. | Big damage boost, HP cost. |
| **Glam** | Replay 1 (once per combat). | Card plays twice first time. |
| **Instinct** | Reduce energy cost by 1. | Makes expensive cards playable. |
| **Momentum** | Damage increases each time played this combat. | Scaling damage. |
| **Nimble** | Gain additional Block. | Simple block boost. |
| **Perfect Fit** | Goes to top of draw pile instead of shuffling in. | Consistent access. |
| **Royally Approved** | Innate + Retain. | Start with it, keep it. Very strong. |

## Ironclad-Specific Notes

**Key Synergy Keywords:**
- **Exhaust** → Triggers Dark Embrace (draw), Feel No Pain (block), Charon's Ashes (AoE damage)
- **Self-damage** → Triggers Centennial Puzzle (draw 3), Rupture (strength), Demon Tongue (negate)
- **Vulnerable** → Triggers Vicious (draw), increases damage by 50% (75% with Paper Phrog)

**Infinite Loop Requirements:**
- Energy generation (Bloodletting, Offering, Expect to Fight)
- Draw (Pommel Strike, Pillage, Burning Pact)
- Zero/low cost attacks (Spite, Anger)
- Exhaust to thin deck (Stoke, Second Wind)
