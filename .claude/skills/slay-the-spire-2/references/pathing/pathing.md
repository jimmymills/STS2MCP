# Map Pathing Guide

> See also: [Strategy Guide](../strategy/strategy-guide.md) for act-by-act checkpoints and combat strategy.

> Poor pathing is the #1 reason new players die in Act 1.

## Reading the Map (In Order)
1. **Check the boss** — Know what you're building toward
2. **Count late elites** — Elites after the treasure chest line are easier (you're stronger)
3. **Count total elites** — Aim for 2-3 minimum; 1 elite = failure
4. **Find rest sites** — Want one before first elite to upgrade
5. **Locate shops** — Early shop = damage spike; Late shop = spend accumulated gold

## Elite Pathing Rules
| Elites in Act | Assessment |
|---------------|------------|
| 1 elite | **Failure** — not enough relics to scale |
| 2 elites | Acceptable minimum |
| 3 elites | Good — standard target |
| 4 elites | Excellent — if you can handle it |

## Path with Options (Critical!)
**Good path:** Branches let you choose elite OR rest site when you arrive
- If struggling → take rest site
- If strong → fight elite
- Still have options for later elites

**Bad path:** Single line of combat → combat → elite with no escape
- If you're struggling, you just die
- No way to recover

> **Example:** If your path goes combat → fire → elite, but you can also branch to combat → fire → shop, you have OPTIONS. Build in fail-safes.

## Fire Between Elites
- Always try to hit a rest site before your first elite
- Upgrading a damage card (12 → 18 damage) makes elites significantly easier
- Fire between elites lets you upgrade OR heal based on need

## Act 1 Pathing Priority
1. Pick path with most elites you can safely fight (2-3 target)
2. Ensure a rest site before first elite
3. Prefer branching paths for flexibility
4. Early shop for damage cards if available

## Act 2 Pathing Priority
1. More aggressive elite hunting — your deck is stronger
2. Shop priority rises (gold accumulated from Act 1)
3. Rest sites for upgrading key scaling cards
4. Avoid dead-end paths with no recovery options

## Act 3 Pathing Priority
1. Final boss preparation — ensure deck is complete
2. Rest sites for healing before boss
3. Elite fights only if reward outweighs risk
4. Shop for final key relics/potions

## Node Types Quick Reference
| Node | Risk | Reward |
|------|------|--------|
| Combat | Low-Medium | Cards, gold, potions |
| Elite | High | Relics (game-changing) |
| Rest Site | None | Heal or upgrade |
| Shop | None | Buy cards/relics/potions |
| ? Event | Variable | Random — can be great or terrible |
| Treasure | None | Free relic |
